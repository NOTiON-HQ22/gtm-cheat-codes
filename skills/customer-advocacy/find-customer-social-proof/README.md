# Find Customer Social Proof

## Use When

Use this when a marketer or seller needs the right customer example, quote, stat, slide, logo, or peer story for a campaign, landing page, sales deck, email, or prospect conversation.

## What It Does

It searches approved customer-proof sources, ranks the best matches, and returns what can be used, where it can be used, and what still needs approval.

## Inputs

- Campaign, content, or sales request
- Target audience, segment, industry, or use case
- Approved customer content library
- Optional CRM or account context
- Usage restrictions

## Outputs

- Ranked customer-proof matches
- Recommended use for each proof point
- Approval and restriction notes
- Source references
- Follow-up questions

## Setup

1. Connect the approved customer content library and optional CRM context.
2. Map the fields in `SCHEMA-MAP.md`.
3. Run `ONBOARDING-PROMPT.md` before the first real proof search.
4. Define who can approve customer names, quotes, metrics, and logos for each channel.

## Safety

Do not use customer proof externally unless it is approved for that exact channel, audience, and asset type.
