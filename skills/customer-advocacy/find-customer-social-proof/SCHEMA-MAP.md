# Schema Map: Find Customer Social Proof

| Field | Description |
| --- | --- |
| `request_type` | Campaign, page, deck, email, event, sales conversation, or other use. |
| `audience` | Target audience or persona. |
| `industry` | Industry filter. |
| `segment` | Customer segment or company size. |
| `use_case` | Workflow, product area, or problem the proof should support. |
| `asset_title` | Name of the customer asset. |
| `asset_type` | Case study, quote, stat, deck slide, logo, call clip, or brief. |
| `asset_url` | Approved source or asset reference. |
| `company_name` | Customer company name, if approved for use. |
| `speaker_name` | Speaker name, if approved for use. |
| `speaker_title` | Speaker title, if approved for use. |
| `approved_for_marketing` | Whether the proof can be used externally in marketing. |
| `restriction_notes` | Channel, audience, expiration, legal, or customer restrictions. |
| `freshness` | Date or recency signal. |
