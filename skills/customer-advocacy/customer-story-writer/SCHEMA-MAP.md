# Schema Map: Customer Story Writer

| Field | Description |
| --- | --- |
| `story_request` | What the requester needs and why. |
| `story_format` | Narrative story or metric-forward story page. |
| `approved_sources` | Links or references to approved transcripts, notes, briefs, and proof points. |
| `customer_profile` | Company, segment, industry, persona, and product context approved for use. |
| `transcript_or_notes` | Interview text or notes to use as source material. |
| `approved_quotes` | Quotes approved for this asset or pending approval. |
| `approved_metrics` | Metrics approved for this asset or pending approval. |
| `usage_rights` | Logo, name, quote, metric, and channel restrictions. |
| `reviewers` | People or teams who must review before external use. |
| `output_destination` | Where the draft or final asset should go. |
