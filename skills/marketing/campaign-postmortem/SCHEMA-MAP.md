# Schema Map: Campaign Postmortem

| Field | Description |
| --- | --- |
| `campaign_name` | Campaign name. |
| `date_range` | Launch and measurement period. |
| `goal` | Stated campaign goal. |
| `audience` | Target audience or segment. |
| `channels` | Channels included in the campaign. |
| `source_docs` | Briefs, plans, docs, meeting notes, and feedback sources. |
| `performance_data` | Approved metrics, dashboard exports, or reports. |
| `timeline_events` | Launch, decision, issue, and follow-up events. |
| `owner_feedback` | Team input and reviewer notes. |
| `known_caveats` | Missing data, attribution limits, or conflicting evidence. |
| `actions` | Follow-up actions, owners, and dates when available. |
